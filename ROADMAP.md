# Roadmap

1. Prove structural compatibility with Taproot's first browser-safe protocol release.
2. Keep the exact-tarball consumer fixture representative of supported public entry points.
3. Validate the Workshop-shaped plugin contract without creating a Workshop dependency.
4. Preserve accessible components and runtime-neutral public export graphs.
5. Stabilize the 1.0 API from consumer feedback without crossing persistence, Site assembly, or installer boundaries.
